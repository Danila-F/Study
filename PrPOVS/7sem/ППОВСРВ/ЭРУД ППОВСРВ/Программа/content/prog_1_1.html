Раздел 1. Технологии и инструментальные системы программирования
проблемно-ориентированных вычислительных средств реального времени

1.1

Инструментальные системы программирования проблемно-ориентированных вычислительных средств

Цели и задачи дисциплины.
Понятие инструментальной системы программирования проблемно-ориентированных вычислительных средств. Назначение и функции инструментальной системы программирования цифровых процессоров обработки сигналов на примере инструментальной системы программирования фирмы Texas Instruments Code Composer Studio.
Основные возможности, функционал и эволюция Code Composer Studio.
Технологический процесс разработки программного обеспечения с использованием Code Composer Studio. Основные подсистемы Code Composer Studio: C/C++ компилятор, оптимизатор линейного ассемблерного кода, транслятор языка ассемблера, компоновщик, архиватор, утилита создания библиотек, формирователь списка перекрестных ссылок, формирователь загрузочных файлов для программаторов, симулятор, отладчик, система визуализации отладочной информации.

1.2

Технология создания программ на ассемблере

Модели распределения памяти, используемые при разработке программ для цифровых процессоров обработки сигналов.
Структура и сегментирование программы на ассемблере. Основные элементы, структура предложения, основные группы директив языка ассемблер.
Использование алгебраического представления ассемблерного кода.
Использование макросредств языка ассемблера. Директивы макроассемблера.
Использование стандартных библиотек.

1.3

Технология создания программ на языке С/С++

Стандарт языка ANSI С/С++. Основные элементы, типы данных, конструкции языка С/С++.
Создание программы с использованием языка С/С++.
Создание многофайловых проектов. Использование и создание ассемблерных вставок в проекте. Использование заголовочных файлов *.h для подключения стандартных библиотек.
Оценка эффективности использования языка С/С++.

Раздел 2. Программная реализация основных алгоритмов цифровой обработки сигналов: формирование сигналов,
цифровая фильтрация, быстрое преобразование Фурье

2.1

Программирование синусоидальных сигналов аналитическими и табличными методами

Цели и задачи дисциплины. Методы программирования синусоидальных сигналов. Аналитические методы программирования синусоидальных сигналов.
Табличные методы программирования синусоидальных сигналов. Прямой и интерполяционный табличные методы программирования синусоидальных сигналов.
Алгоритмы и программы для формирования гармонических и полигармонических процессов с изменяющейся частотой.

2.2

Программирование цифровых фильтров

Программирование цифровых нерекурсивных фильтров. Распределение памяти данных и коэффициентов. Организация циклов.
Программирование цифровых рекурсивных фильтров (ЦРФ). Распределение памяти данных и коэффициентов. Прямое программирование ЦРФ.
Параллельное и последовательное соединение цифровых рекурсивных фильтров. Распределение памяти данных. Циклическое программирование ЦРФ.
Анализ точностных характеристик цифровых фильтров. Квантование передаточной функции дискретной системы.
Анализ чувствительности цифровых фильтров. Ошибки квантования и округления при вычислениях в цифровом фильтре.

2.3

Программирование базовых операций алгоритмов быстрого преобразования Фурье (БПФ)

Программная реализация различных типов базовых операций алгоритмов БПФ для процессоров семейства TMS320. Использование команд умножение с накоплением.
Частные случаи реализация базовой операции БПФ.

2.4

Программирование алгоритмов БПФ с прореживанием по времени

Организация двоично-инверсного счетчика для выполнения операции двоично-инверсной перестановки данных.
Программная реализация алгоритма БПФ по модулю два с замещением, прореживанием по времени и прямым порядком входных данных.
Программная реализация алгоритма БПФ по модулю два с замещением, прореживанием по времени и двоично-инверсным порядком входных данных.

2.5

Программирование алгоритмов БПФ с прореживанием по частоте

Программная реализация алгоритма БПФ по модулю два с замещением, прореживанием по частоте и прямым порядком входных данных.
Программная реализация алгоритма БПФ по модулю два с замещением, прореживанием по частоте и двоично-инверсным порядком входных данных.
Программная реализация алгоритма БПФ с поблочно-плавающей запятой. Основные преимущества данного метода.
Программная реализация алгоритма векторного БПФ. Одновременная обработка нескольких процессов. Организация памяти данных и циклов для векторного БПФ.

Раздел 3. Программирование различных задач цифровой обработки сигналов на проблемно-ориентированных процессорах

3.1

Организация вычислений с плавающей запятой (точкой) на процессорах использующих арифметико-логические устройства с фиксированной запятой (точкой)

Стандарт IEEE754.
Реализация операций с плавающей запятой (ПЗ) на процессорах использующих арифметико-логические устройства с фиксированной запятой (ФЗ). Представление чисел в форматах с ПЗ. Основные форматы чисел с ПЗ, используемые в микропроцессорных устройствах.
Реализация операции сложения чисел с ПЗ. Выравнивание порядков и сложение мантисс. Реализация вычитания с ПЗ.
Реализация операции умножения чисел с плавающей запятой. Сложение порядков и умножение 24-разрядных мантисс.
Реализация операции деления чисел с плавающей запятой. Вычитание порядков и деление 24-разрядных мантисс.
Реализация операции извлечения квадратного корня из чисел представленных в форме с ПЗ.
Реализация операции вычисления функции арктангенса из чисел представленных в форме с ПЗ.

3.2

Использование проблемно-ориентированных процессоров для компрессии аудиоданных

Компрессия данных. Кодеки. Кодек G711.
Использование процессоров семейства TMS320 для компрессии сигналов. Сжатие и восстановление сигналов с использованием метода μ-255 (u-Law).
Сжатие и восстановление сигналов с использованием метода A-Law.
Псевдологарифмический масштаб данных.

3.3

Использование проблемно-ориентированных процессоров для усреднения сигналов.

Программная реализация алгоритма линейного усреднения для проблемно-ориентированных процессоров семейства TMS320.
Программная реализация алгоритма экспоненциального усреднения для проблемно-ориентированных процессоров семейства TMS320.
Программная реализация алгоритма усреднения методом скользящего среднего с простым и взвешенным сглаживанием для проблемно-ориентированных процессоров семейства TMS320.
Когерентное и некогерентное усреднение.
Особенности усреднения сигналов в частотной области.

3.4

Формирование случайных процессов с использованием проблемно-ориентированных процессоров.

Случайные числа. Случайные и псевдо случайные последовательности.
Формирование псевдослучайных последовательностей. М-последовательности. Топология обратной связи сдвигового регистра для формирования последовательностей различной длины.
Формирование случайных процессов с заданной спектральной плотностью мощности.
Полигармонические и псевдослучайные процессы.
