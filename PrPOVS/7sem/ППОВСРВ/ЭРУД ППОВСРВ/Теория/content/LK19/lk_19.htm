Реализация операции деления чисел с плавающей запятой

Алгоритм деления чисел с плавающей запятой

Основу алгоритма операции деления составляет разложение частного от деления мантисс в ряд с определенной точностью и вычитание порядков.
Тогда алгоритм умножения чисел f1 = m1 · be1 и f2 = m2 · be2 может быть записан следующим образом:
f1 · f2 = m1 / m2 · b(e1–e2)
Для пояснения данного алгоритма представим мантиссы в следующем виде:
делимое –  A = AHI + ε·ALO
делитель – B = BHI + ε·BLO
где ε = 1/2длина младшей части = 1/216
Тогда процесс деления может быть описан выражением
 
 

Обозначив первое частное через новую переменную Q, которая может быть записана как QHI+ε·QLO и учитывая что ε2 << 0, получим окончательное выражение для алгоритма деления

Алгоритм извлечения квадратного корня из чисел с плавающей запятой

Множество математических операций над комплексными числами значительно легче осуществляется, если числа представлены в тригонометрической форме. К ним относятся в частности операции умножения, деления, извлечения корня квадратного.
Корнем n-ой степени из комплексного числа z называется такое комплексное число w, n-я степень которого равна z, то есть . Корень n-ой степени из комплексного числа обозначается символом и на множестве комплексных чисел имеет ровно n значений. Если комплексное число задано в тригонометрической форме: , то все значения корня n-ой степени вычисляются по формуле Муавра (Абрахам де Муавр (1667 - 1754) - английский математик):



Геометрически все значения корня лежат на окружности радиуса с центром в начале координат и образуют правильный n - угольник.
В случае извлечения корня квадратного из комплексного числа в общем случае будут получены два корня, которые расположены на одном отрезке:


Важной задачей, возникающей при создании программ, реализующих алгоритмы цифровой обработки сигналов, является задача преобразования комплексных чисел из алгебраической формы представления в тригонометрическую форму представления и наоборот. Для решения этих задач требуется обеспечить возможность вычисления функций извлечения корня квадратного и арктангенса. Алгоритмы вычисления функций синуса и косинуса были рассмотрены в предыдущих лекциях.
Учитывая тот факт, что скорость выполнения операций сложения, вычитания и умножения в процессорах цифровой обработки сигналов одинакова, можно использовать алгоритм класса цифра за цифрой, заключающийся в том, что результат получается в процессе выполнения последовательных итераций. Значение подкоренного выражения находится в диапазоне [0,25 .. 1[. Принимаем значение первой итерации корня квадратного равным весу старшего значащего разряда равного 0,5. Далее возводим значение первой итерации корня квадратного в квадрат и сравниваем с подкоренным выражением. Если полученный результат больше чем реальное значение подкоренного выражения, то соответствующий разряд в итерации обнуляем. Если полученный результат меньше или равен реальному подкоренному выражению, то соответствующий разряд в итерации оставляем установленным в единицу. Далее приступаем к следующей итерации, добавив в подкоренное выражение вес следующего, более младшего разряда, значение которого равно 0,25. В результате выполнения каждой итерации мы определяем значение следующего разряда искомого результата вычисления корня квадратного. Таким образом, появляется возможность выполнения вычислений для определения результата с требуемой точностью.

Алгоритм для вычисления корня квадратноого в форме с плаваюшей запятой представлен на рисунке 3.1.6. Используется формат данных float (32 разрядное представление числа с плавающей запятой согласно стандарта IEEE754).

Алгоритм вычисления функции арктангенса из чисел представленных в форме с плавающей запятой

Быстрые и точные методы вычисления арктангенса отношения мнимой и действительной частей комплексного числа Z = P + i Q исследовались весьма активно, так как оценка аргумента  комплексного значения находит множество применений в цифровой обработке сигналов. Аргумент φ комплексного числа Z определяется как φ = arctg (Q / P).
Основной сложностью вычисления функции арктангенса является его высокая нелинейность в окрестности значения функции φ стремящемуся к значениям π/2 и - π/2 с обеих сторон.
Максимальной скорости вычисления арктангенса можно добиться используя табличные методы вычисления, в которых значение Q / P задает адрес ячейки ПЗУ, содержащей значение аргумента φ.
Для того чтобы уйти от необходимости вычисления функции arctan в диапазонах большого значения аргумента функции φ, можно использовать свойство зеркальной симметрии функций arctan и arccotan относительно точек π/4 и -π/4, которое сводится к следующим формулам 3.1.1:
