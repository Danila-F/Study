Практика
1. Практические занятия.
1.1 Программирование синусоидальных сигналов с использованием прямого табличного метода. 1.2 Программирование синусоидальных сигналов с использованием интерполяционного табличного метода. 1.3 Расчет полной мощности синусоидального сигнала и мощности основной гармоники для сравнения точности прямого и интерполяционного методов программирования синусоиды. 1.4 Расчет мощности паразитных гармоник в сформированном сигнале и коэффициента нелинейных искажений. 1.5 Программирование цифрового нерекурсивного фильтра с использованием циклов и различных методов реализации задержки. 1.6 Программирование двоично-инверсного счетчика для алгоритма БПФ, перестановка данных. 1.7 Использование алгоритма компрессии данных μ-Law для перевода данных в логарифмический масштаб. 1.8 Использование алгоритма компрессии данных A-Law для перевода данных в логарифмический масштаб.
2. Лабораторные работы.
2.1. Программирование синусоидальных сигналов методом суммы двух углов. 2.2. Программирование синусоидальных сигналов методом двойного угла. 2.3. Программирование цифрового рекурсивного фильтра. 2.4. Измерение амплитудно-частотной характеристики цифрового рекурсивного фильтра. 2.5. Программирование базовой операции алгоритмов быстрого преобразования Фурье. 2.6. Программирование быстрого преобразования Фурье с прореживанием по времени. 2.7. Программирование быстрого преобразования Фурье с прореживанием по частоте. 2.8. Измерение амплитудно-частотной характеристики цифрового рекурсивного фильтра.
