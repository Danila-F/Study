Примерный перечень задач к контрольной работе №1 по учебной дисциплине ППО ВС РВ

 

Написать головную программу и подпрограмму на Ассемблере для преобразования данных, вводимых из одного пространства памяти процессора TMS320VC5402,
и выводом результата в другое адресное пространство.

Каждая директива Ассемблера и каждая команда должны сопровождаться комментариями.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Написать головную программу и подпрограмму для TMS320VC5402 для поиска максимального значения в массиве дробных чисел со знаком. Массив вводится из параллельного порта 7 (PA7), число элементов массива COUNT находится в ячейке памяти данных и передается в подпрограмму через регистр AR2, максимальное значение записывается в память данных по адресу MAX.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Написать головную программу и подпрограмму для TMS320VC5402 для поиска минимального значения в массиве целых чисел без знака. Массив вводится из последовательного порта McBSP0, число элементов массива NUMB задается константой и передается в подпрограмму через регистр AR7, минимальное значение записывается в память данных по адресу MIN.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. Написать головную программу и подпрограмму для TMS320VC5402 для изменения знака элементов массива целых чисел на противоположный. Адреса исходного OLD и результирующего NEW массивов передаются в подпрограмму через регистры AR3 и AR4, число элементов массива ELEM задается константой и передается в подпрограмму через регистр AR6.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

4. Написать головную программу и подпрограмму для TMS320VC5402 для изменения знака элементов массива дробных чисел на противоположный знак. Массив вводится из параллельного порта 22 (PA22), число элементов массива CHIS находится в ячейке памяти данных и передается в подпрограмму через регистр BRC, измененные значения выводятся в последовательный порт McBSP1.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

5. Написать головную программу и подпрограмму для TMS320VC5402 для инвертирования всех разрядов элементов массива чисел. Адрес исходного массива данных MASIV передается в подпрограмму через регистр AR1, число элементов массива CNT находится в ячейке памяти данных и передается в подпрограмму через регистр AR3, новые значения выводятся в последовательный порт McBSP0.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

6. Написать головную программу и подпрограмму для TMS320VC5402 для инвертирования всех разрядов элементов массива чисел. Массив вводится из последовательного порта McBSP1, число элементов массива NUM задается константой и передается в подпрограмму через регистр AR5, адрес выходного массива данных INV передается в подпрограмму через регистр AR6.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

7. Написать головную программу и подпрограмму для TMS320VC5402 для инвертирования знакового разряда элементов массива целых чисел. Массив вводится из параллельного порта 15 (PA15), число элементов массива COUNT задается константой и передается в подпрограмму через регистр BRC, измененные значения выводятся в последовательный порт McBSP0.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

8. Написать головную программу и подпрограмму для TMS320VC5402 для инвертирования знакового разряда элементов массива дробных чисел. Массив вводится из последовательного порта McBSP0, число элементов массива ELEM находится в ячейке памяти данных и передается в подпрограмму через регистр AR4, измененные значения выводятся в параллельный порт 12 (PA12).

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

9. Написать головную программу и подпрограмму для TMS320VC5402 для прибавления к элементам массива беззнаковых чисел константы KONST. Массив вводится из последовательного порта McBSP1, число элементов массива RAZM задается константой и передается в подпрограмму через регистр BRC, адрес выходного массива данных SUMM передается в подпрограмму через регистр AR4.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

10. Написать головную программу и подпрограмму для TMS320VC5402 для вычитания из элементов массива чисел со знаком содержимого ячейки DAT. Массив вводится из параллельного порта 20 (РА20), число элементов массива CHIS находится в ячейке памяти данных и передается в подпрограмму через регистр AR3, измененные значения выводятся в последовательный порт McBSP1.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

11. Написать головную программу и подпрограмму для TMS320VC5402 для умножения элементов массива целых чисел на дробную константу CONS. Адрес исходного массива данных DAT передается в подпрограмму через регистр AR4, число элементов массива ELEM задается константой и передается в подпрограмму через регистр AR3, новые значения выводятся в параллельный порт 34 (PA34).

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

12. Написать головную программу и подпрограмму для TMS320VC5402 для умножения элементов массива дробных чисел на целое число из ячейки MUL. Массив вводится из параллельного порта 14 (PA14), число элементов массива COUNT находится в ячейке памяти данных и передается в подпрограмму через регистр AR1, измененные значения выводятся в последовательный порт McBSP0.

13. Написать головную программу и подпрограмму для TMS320VC5402 для умножения элементов массива дробных чисел на дробное число из ячейки COEF. Адрес исходного массива данных DAT передается в подпрограмму через регистр AR4, число элементов массива CNT задается константой и передается в подпрограмму через регистр BRC, новые значения выводятся в параллельный порт 30 (PA30).
