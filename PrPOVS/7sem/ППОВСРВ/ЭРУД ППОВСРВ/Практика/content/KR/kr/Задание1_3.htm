Задания по программированию алгоритма БПФ к контрольной работе №1 по учебной дисциплине ППО ВС РВ

 

Написать программу БПФ, отладить ее и проверить ее работу на трех стандартных сигналах: импульс (первое значение 7FFF16, остальные 0), постоянная (все значения 7FFF16) и синусоида (заданная гармоника) в соответствие с вариантом из таблицы. Представить листинг программы с комментариями и результаты работы программы в виде трех пар графиков изображения Фурье (действительная и мнимая часть).

Внимание!

1) При написании программы использовать методичку по лабораторным работам:

Программирование проблемно-ориентированных вычислительных средств реального времени : лаб. практикум для студ. спец. 1-40 02 02 «Электронные вычислительные средства» днев. формы обуч. В 2 ч. Ч. 2 / В. Б. Клюс, М. В. Качинский, А. Б. Давыдов. – Минск : БГУИР, 2011.

2) При написании программы можно использовать прилагаемые таблицы функции синуса:

а) на 128 отсчетов (файл SIN128.TAB);

б) на 256 отсчетов (файл SIN256.TAB);

в) на 512 отсчетов (файл SIN512.TAB).
