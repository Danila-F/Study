2) Рекомендации по программированию
Для вычисления полной энергия сигнала можно использовать команду вычисления суммы квадратов, а для повышения точности не выполнять масштабирование. Наличие в аккумуляторе 8 защитных разрядов позволяет просуммировать без переполнения 256 квадратов, а с выключенным битом FRCT - 512.
Для вычисления энергия основной гармоники можно использовать команду умножения со сложением и второй аккумулятор, а для повышения точности не выполнять масштабирование, как и для полной мощности. В этом случае полная мощность и мощность основной гармоники будут находиться в двух аккумуляторах в одинаковом масштабе.
