# Библиотека компонентов для Altium Designer
## Правила оформления библиотеки
* Единицы измерения - мм.
* Использовать шрифт ГОСТ тип А из файла "ГОСТ тип А.ttf", который можно найти в репозитории.
* Размер кегля - 10 для нумерации пинов, 14 для всего остального.
* Сетка УГО компонентов - 2,5 мм.

### Как пользоваться библиотекой

* В случае, если у вас нет гит-клиента:

1.  Скачать архив с файлами библиотеки, нажав кнопку "Clone or download", расположенную справа над списком файлов репозитория, и выбрав пункт "Download ZIP".
2.  Разархивировать скачанные файлы в папку, путь к которой не содержит национальных символов и пробелов.
3.  Запустить Altium Designer.
4.  Открыть панель Libraries, нажать кнопку "Libraries...", выбрать вкладку "Installed".
5.  Нажать на кнопку "Install...", из выпадающего списка выбрать "Install from file...".
6.  В окне поиска из выпадающего списка выбрать расширение файла "*.DBLIB".
7.  В корневой папке библиотеки найти и выбрать файл "GOST.DbLib".
8.  Нажать "Открыть" или "Open".
9.  Нажать "Close".
10. Пользоваться : )

* В случае, если у вас есть гит-клиент:

1. Нажать кнопку "Clone or download", скопировать появившуюся ссылку на репозиторий для гит-клиента.
2. Клонировать репозиторий через гит-клиент на локальную машину.
3. Установить библиотеку, повторив пункты 3 -- 10 из предыдущего списка.
